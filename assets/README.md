# Assets — Kenworth Engineering

Drop any photography, video, logos, brochures, or other client-supplied media here. The designer (phase 4) and developer (phase 5) read this folder for real imagery to embed.

Structure:
- `photography/` — shop-floor photos, machine photos, named-person portraits, machined-part details
- `video/` — any video clips for hero or capability sections
- `logos/` — vector SVG when available, plus any sub-marks or alt versions
- `docs/` — capability sheet PDFs, ISO certificate, vendor approval pack, anything print-ready

Notes:
- Prefer .webp for photography (with .jpg fallback) and .svg for logos/marks.
- Keep file names lowercase-with-hyphens.
- Hero images target ≤ 200KB. Card images target ≤ 80KB.
